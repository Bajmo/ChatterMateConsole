# ChatterMateConsole
ChatterMateConsole est un mini-projet Visual Studio 2022 réalisé en C++ dont le but était de créer une application console de chat/journal permettant à 10 personnes différentes de communiquer sur la même machine en correspondance avec la personne qui l'utilise à un moment donné.
